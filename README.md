# Hello World Python Program

I am building foundational programming skills to strengthen my software engineering expertise. This project shows my commitment to learning coding practices and building a professional workflow.

This is a simple "hello world" program written in Python. It prints a greeting message to the console and serves as a demonstration of a working development environment. It includes basic print statement, variable usage, and a personal practice section for an arithmetic operation.

My purpose for creating this software is to build a functional development environment and practice the fundamentals of python programming. This also serves as a starting point for building more complex software projects in the future. I aim to develop good coding habits, including proper documentation, version control, and clean code structure.
